# Country Data with Flags JSON

This repository contains a comprehensive dataset of countries that merges standard geographic data (ISO codes, capitals, regions, states) with their corresponding emoji flags.

The main file, `countries-and-states.json`, provides a list of country objects. The data was generated by merging country metadata with emoji flag datasets, ensuring that the **flag is the primary key** for easy visual scanning.

### File Format: `JSON`

Each entry in the JSON array represents a single country and adheres to the following structure:

| Key | Type | Description | Example |
| :--- | :--- | :--- | :--- |
| **`flag`** | `String` | The Unicode emoji flag for the country. | `"ğŸ‡¦ğŸ‡«"` |
| **`name`** | `String` | The common English name of the country. | `"Afghanistan"` |
| **`code2`** | `String` | ISO 3166-1 alpha-2 country code. | `"AF"` |
| **`code3`** | `String` | ISO 3166-1 alpha-3 country code. | `"AFG"` |
| **`dialcode`** | `String` | The international calling code. | `"+93"` |
| **`capital`** | `String` | The capital city of the country. | `"Kabul"` |
| **`region`** | `String` | The continental region. | `"Asia"` |
| **`subregion`** | `String` | The specific sub-geographic region. | `"Southern Asia"` |
| **`states`** | `Array` | A list of states/provinces/subdivisions. | `[{...}]` |
